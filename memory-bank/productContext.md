# Product Context: Money Pink

## Overview
**Money Pink** is a high-fidelity, visually rich financial planning and tracking dashboard designed for a family or small collaborative group. It provides a central, real-time interface to monitor investments, member contributions, and outstanding financial states.

The application is styled with a custom **Liquid Glassmorphism Theme (Pink edition)**, utilizing HSL-tailored colors, interactive glass cards, animated background liquid blobs, and responsive charts.

## Core Product Pillars & Features

### 1. Real Estate (อสังหาริมทรัพย์) Dashboard
Focuses on tracking physical or long-term asset investments. It tracks:
* **ต้นทุน (Cost)**: Total capital requirements or target investment amount.
* **ยอดจ่าย (Paid)**: Amount actually paid by members.
* **ค้างจ่าย (Outstanding Pay)**: Remaining payment obligations (Cost - Paid).
* **รายได้ (Income)**: Earnings or rental returns generated by the assets.
* **ยอดรับ (Received)**: Income actually received or distributed.
* **ค้างรับ (Outstanding Receive)**: Uncollected earnings (Income - Received).

### 2. Central Money (เงินกลาง) Dashboard
Tracks shared treasury funds, operations, and short-term loans/deposits. It tracks:
* **ยอดเรียก (Called)**: Funding requests made to members.
* **ยอดเก็บ (Collected)**: Contributions gathered from members.
* **ยอดค้าง (Outstanding)**: Unpaid funding requests.
* **ยอดเบิกเงิน (Withdrawn)**: Operational or joint expenses paid out.
* **ยอดยืมเงิน (Borrowed)**: Short-term personal loans taken by members from the treasury.
* **ยอดคืนเงิน (Returned)**: Loan repayments made back to the treasury.
* **ยอดค้างคืน (Outstanding Return)**: Remaining unpaid loans.
* **เงินคงเหลือ (Treasury Balance)**: Calculated dynamically as `(Collected + Returned) - (Withdrawn + Borrowed)`.

### 3. Key Interactive Capabilities
* **Dynamic Filtering**: Filter dashboards by **Year (ปี)** and **Member (สมาชิก)**. Supports "All" selection for aggregated multi-year or group insights.
* **Interactive Charting**: Visualized using `Recharts` for stacked bars, line charts, and comparison views.
* **Export & Capture**:
  * **Capture Screenshot**: Generates a high-quality `.png` report using `html2canvas`.
  * **Copy Image**: Copies the dashboard visual representation directly to the user's system clipboard using `navigator.clipboard`.
  * **Smooth HMR & Hot-Reloading**: Real-time hot updates for swift adjustments.
